Query 3 - Method untuk Menghitung Pengembalian Pinjaman


    def hitung_pengembalian(self, nama_anggota, tahun_pengembalian):
        for dict_anggota in self.anggota:
            if dict_anggota["nama_anggota"] == nama_anggota:
                jumlah_pinjaman = dict_anggota["jumlah_pinjaman"]
        
        total_pengembalian = jumlah_pinjaman + (jumlah_pinjaman * 0.05) * tahun_pengembalian
        print(f"Total pengembalian dari {nama_anggota} selama {tahun_pengembalian} Tahun sebesar (Rp.): {int(total_pengembalian):,}")

Penjelasan:

    Method hitung_pengembalian untuk menghitung jumlah dana yang harus dibayarkan oleh anggota UMKM berdasarkan tahun pembayaran dengan margin 5% per tahun.

Do's:

  • Menerima input nama_anggota dan tahun_pengembalian
  • Melakukan looping dari variabel anggota untuk mengambil nama anggota yang dipilih dan jumlah pinjaman nya
  • Melakukan pengecekan nama anggota yang dipilih menggunakan if

Query 6 - Method untuk Menghitung Keuntungan Koperasi


    def hitung_keuntungan(self):
        transaksi_beli = 0
        transaksi_jual = 0
        for items in self.transaksi:
            if items["jenis_transaksi"] == "jual":
                transaksi_jual = transaksi_jual + items["jumlah_transaksi"]
            elif items["jenis_transaksi"] == "beli":
                transaksi_beli = transaksi_beli + items["jumlah_transaksi"]

        total_keuntungan = transaksi_jual - transaksi_beli
        return total_keuntungan

Penjelasan:

    Method hitung_keuntungan untuk menghitung total keuntungan dari transaksi koperasi yaitu transaksi jual dikurangi beli.

Do's:

  • Menginisiasi variabel transaksi_beli dan transaksi_jual agar bisa digunakan saat looping karena akan terus menambahkan nominal transaksi tiap jenis nya ke dalam dua variabel tersebut
  • Melakukan looping dari variabel transaksi untuk mengambil data transaksi tiap anggota berdasarkan jenis transaksi nya dan ditambahkan ke variabel transaksi_beli dan transaksi_jual
  • Menghitung total keuntungan koperasi berdasarkan keseluruhan nominal transaksi jual dikurangi transaksi beli

Don'ts:

  • Jika tidak menginisiasi variabel transaksi beli dan jual di awal akan terkena error karena variabel tersebut belum di inisiasi terlebih dahulu

Query 8 - Method untuk Mencatat Data Bank Sampah


    def catat_sampah(self, nama_anggota, jenis_sampah, jumlah):        
        if nama_anggota in self.data_sampah:
            for i in self.data_sampah:
                if i == nama_anggota:
                    if jenis_sampah in self.data_sampah[i]:
                        self.data_sampah[nama_anggota][jenis_sampah] = self.data_sampah[nama_anggota][jenis_sampah] + jumlah
                    else:
                        self.data_sampah[nama_anggota][jenis_sampah] = jumlah
        else:
            self.data_sampah[nama_anggota] = {jenis_sampah: jumlah}

Penjelasan:

    Method catat_sampah untuk pencatatan data sampah, jenis, dan jumlah nya yang telah dikumpulkan setiap anggota

Do's:

  • Menerima input nama_anggota, jenis_sampah, dan jumlah
  • Melakukan looping dari variabel data sampah untuk mengambil data pengumpulan sampah berdasarkan nama_anggota dan jenis_sampah
  • Melakukan pengecekan apakah nama anggota dan jenis sampah yang di input sudah ada dalam data_sampah, jika belum ada maka akan menambahkan data baru, jika sudah ada maka akan mengupdate data nya

Don'ts:

  • Jika tidak ada pengecekan terkait nama dan jenis sampah yang telah ada, maka data nya akan bertambah terus dan akan menjadi kurang rapih

Query 9 - Method untuk Menghitung Nilai Tukar Sampah


    def hitung_nilai_tukar(self):
        total_per_anggota = {}
        total_nilai = 0
        for items in self.data_sampah:
            nilai_anggota = 0
            for type in self.data_sampah[items]:
                if type == "plastik":
                    nilai_anggota = nilai_anggota + (5000 * self.data_sampah[items][type])
                elif type == "kertas":
                    nilai_anggota = nilai_anggota + (2000 * self.data_sampah[items][type])
            total_nilai = total_nilai + nilai_anggota
            total_per_anggota[items] = nilai_anggota
            print(f"Total nilai tukar {items} (Rp.): {nilai_anggota:,}")
        print(f"Total nilai tukar semua anggota (Rp.): {total_nilai:,}")

Penjelasan:

    Method hitung_nilai_tukar untuk menghitung nilai tukar berdasarkan data sampah (misalnya plastik Rp. 5000/kg, kertas Rp. 2000/kg, dsb)

Do's:

  • Inisiasi variabel total_per_anggota, total_nilai, dan nilai_anggota untuk placeholder sementara dari hasil penghitungan nilai tukar tiap anggota
  • Melakukan looping dari variabel data_sampah untuk setiap anggota dan dikelompokkan per jenis sampah nya
  • Menghitung nilai tukar sampah berdasarkan jenis sampah dikalikan jumlah dan nilai per kg jenis sampah nya

Don'ts:

  • Pastikan sudah menginisiasi variabel total_per_anggota, total_nilai, dan nilai_anggota agar tidak error

Query 10 - Method untuk Pesan Edukasi Lingkungan


    def pesan_edukasi(self, nama):
        total_sampah = 0
        for items in self.data_sampah:
            if items == nama:
                for jenis_sampah in self.data_sampah[nama]:
                    total_sampah = total_sampah + self.data_sampah[nama][jenis_sampah]
        if total_sampah > 50:
            print(f"\n{nama} telah mengumpulkan {total_sampah} kg sampah.\n"
                  + "Anda adalah pahlawan lingkungan sejati!")
        elif total_sampah > 20:
            print(f"\n{nama} telah mengumpulkan {total_sampah} kg sampah.\n"
                  + "Mari bersama berkontribusi lebih banyak untuk bumi!")
        elif total_sampah > 0:
            print(f"\n{nama} telah mengumpulkan {total_sampah} kg sampah.\n"
                  + "Semoga ini menjadi awal yang baik untuk kita bisa mengumpulkan lebih banyak sampah lagi!")
        else:
            print(f"\n{nama} belum mengumpulkan sampah.\nMari kita mulai mengumpulkan sampah agar Bumi kembali bersih!")

Penjelasan:

    Method pesan_edukasi untuk memberikan pesan yang dipersonalisasi dan disesuaikan dengan pencapaian pengumpulan sampah tiap anggota agar bisa memberikan motivasi.

Do's:

  • Menginisiasi variabel total_sampah untuk melihat total sampah yang dikumpulkan per anggota
  • Melakukan looping dari variabel data_sampah untuk mengambil data sesuai dengan nama anggota yang dipilih
  • Membuat kondisional if else agar bisa memberikan pesan yang berbeda sesuai pencapaian pengumpulan sampah dari anggota yang dipilih

Don'ts:

  • Saat pembuatan kondisional if else nya, mulai dari kriteria lebih besar dahulu baru ke yang kecil agar sistematis dan output yang diinginkan bisa sesuai
  • Kalau dimulai dari yang kecil jika ada data yang masuk ke kriteria besar maka yang akan dijalankan yaitu kondisi kecil nya karena begitu ada yang sesuai dengan kriteria maka yang paling atas akan langsung dijalankan tanpa melihat lagi kondisi dibawah nya

    Program utama yang mengintegrasikan class UMKMSystem, Koperasi, dan BankSampah, beserta method-method di tiap class berdasarkan input dari user secara interaktif sesuai dengan kebutuhan tiap method, disertai juga laporan lengkap dari data yang sudah di isi.

Do's:

  • Membuat method menu yang menerima input nama_umkm, di dalam nya berisikan looping agar aplikasi bisa berjalan secara kontinu sampai user memilih opsi keluar
  • Pada menu UMKM System, dibuat pengecekan apakah UMKM tersebut sudah aktif dalam program Bank Sampah sebagai syarat untuk mendapat pinjaman awal sebesar Rp. 50 juta
  • Menambahkan try dan except general untuk menghandle ketika ada error
  • Menggunakan method isinstance untuk mengecek apakah objek merupakan instance dari class tersebut atau tidak

Don'ts:

  • Jangan lupa menambahkan break pada method menu agar program bisa exit dan tidak infinite loop
